  <h2 style="border-bottom: 1px solid #d8dee4; color: #28303f;"> 👨‍💻 About Me </h2>
  <div align="center" style="margin-bottom: 30px; line-height: 1.6; max-width: 800px;">
    <p>
      결제라는 민감한 흐름 속에서 시스템의 안정성은 곧 사용자의 신뢰와 직결돼요.<br>
      지난 3년간 빌링 솔루션과 백오피스를 운영하며 마주한 수많은 문제들은 기능 구현 이상의 치열한 고민을 안겨주었죠.<br>
      특히 숨 막히게 쏟아지는 대용량 데이터를 다루는 과정은 결코 쉽지 않은 과제였어요.<br>
      메모리 부족의 위협 속에서 청크(Chunk) 단위 배치 프로세스를 설계하여 방대한 트랜잭션을 거뜬히 소화해냈습니다.<br>
      여기에 멈추지 않고 쿼리 성능의 한계를 극복하고자 데이터베이스 깊숙한 곳을 파고들었어요.<br>
      복잡하게 얽힌 비즈니스 로직은 저장 프로시저(Stored Procedure)를 적극 활용해 DB 단에서 최적화하여 처리 속도를 비약적으로 끌어올렸죠.<br>
      수많은 REST API로 외부 시스템과 긴밀하게 소통하고, 실무자들의 답답함을 해소하는 백오피스를 만들어가는 과정.<br>
      그 속에서 얻은 단단한 데이터 핸들링 노하우를 바탕으로 견고한 뼈대를 세우는 개발자로 도약하고 싶어요.
    </p>
  </div>
